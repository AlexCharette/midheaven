MIDHEAVEN

your offline astrology workbench

Annotate a birth chart with astrologer's comments, either manually or from a transcription. If you connect a whisper model, you can transcribe in the app itself. Notes are automatically tagged with the chart elements they refer to.

all releases ☞

first launch · The builds are not notarized with Apple yet. On macOS, open the app once, dismiss the warning, then approve it under System Settings → Privacy & Security (Open Anyway). If macOS instead claims the app is “damaged” (downloads of v0.1.0), clear the quarantine flag in Terminal: xattr -d com.apple.quarantine /Applications/Midheaven.app. On Windows, choose More info → Run anyway when SmartScreen asks.

transcription · Live transcription is local (whisper.cpp) and needs a user-supplied ggml model file — see the README for where to get one, download this one (148 MB download) to get started, or browse for more.

offline · The app makes no network requests: places are embedded and all calculations are done locally. Exported readings are single HTML files that can simply be opened in a browser — or rich PDFs, in a more printer-friendly format.
